## CrossCore_Mod_Transfer_Tool
A simple Mod Transfer tool for game CrossCore/一个简易的交错战线Mod传输工具

# 适用于使用MuMu模拟器，从本地上传Mod文件的用户
原理为使用ADB传输本地文件至模拟器，其他模拟器用户可自行参考修改
